# 🌿 Devocional Diário em Python

Este é um projeto simples desenvolvido em Python com o objetivo de compartilhar mensagens e versículos bíblicos diariamente.

## 💛 Propósito

Criar uma ferramenta simples que possa edificar vidas e, ao mesmo tempo, me ajudar a aprender programação.

## 🚀 Funcionalidades

* Exibe versículos aleatórios
* Mostra mensagens de encorajamento
* Interação com o usuário

## 🛠️ Tecnologias usadas

* Python

## ✨ Em desenvolvimento

Este projeto faz parte do meu processo de aprendizado, e será melhorado com o tempo.
